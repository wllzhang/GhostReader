# 使用方法

## 导入文本

1. 点击活动栏中的 Ghost READER 图标打开阅读列表面板
2. 点击阅读列表标题栏的 "+" 按钮
3. 选择要导入的 TXT 文件

![主界面](/main.png)

## 开始阅读

点击阅读列表中的文本即可开始阅读，内容会显示在状态栏中。

![状态栏显示](/status.png)

## 快捷键

### Reading 模式

在 Reading 模式下，以下快捷键可用：

| 快捷键                          | 功能   |
| ------------------------------- | ------ |
| `Ctrl+Left` (Mac: `Cmd+Left`)   | 上一行 |
| `Ctrl+Right` (Mac: `Cmd+Right`) | 下一行 |

![快捷键设置](/quickkey.png)

### 模式切换

- **切换到 Reading 模式**: 执行命令 `GhostReader: 快捷键激活`
- **切换到 Coding 模式**: 执行命令 `GhostReader: 快捷键禁用`

在 Coding 模式下，阅读快捷键被禁用，不会与编辑器快捷键冲突。

## 跳转到指定位置

执行命令 `GhostReader: 状态栏阅读：跳转` 可以跳转到指定行号。

## 删除文本

在阅读列表中右键点击文本，选择删除图标即可移除文本。
