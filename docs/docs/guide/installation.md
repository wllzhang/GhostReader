# 安装指南

## 从 VS Code 市场安装

1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展面板
3. 搜索 "GhostReader"
4. 点击安装

![安装插件](/main.png)

## 从 VSIX 文件安装

如果你需要安装特定版本，可以从 [GitHub Releases](https://github.com/wllzhang/GhostReader/releases) 下载 `.vsix` 文件：

1. 下载 `.vsix` 文件
2. 在 VS Code 中按 `Ctrl+Shift+P`
3. 输入 "Extensions: Install from VSIX..."
4. 选择下载的文件
