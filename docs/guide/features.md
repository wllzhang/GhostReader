# 功能介绍

GhostReader 提供了一系列实用的阅读功能，让你在工作时也能优雅地MO。

## 核心功能

### 📖 状态栏阅读

将小说内容显示在 VSCode 底部状态栏，既隐蔽又方便。

**特点：**

- 不占用编辑器空间
- 可自定义显示行数
- 支持多行显示

**使用方法：**
在书架中点击书籍即可开始阅读，内容会自动显示在状态栏。

---

### ⌨️ 快捷键操作

通过快捷键快速翻页，提升阅读效率。

**支持的快捷键：**

| 功能   | Windows/Linux | macOS     |
| ------ | ------------- | --------- |
| 上一页 | `Ctrl + ←`    | `Cmd + ←` |
| 下一页 | `Ctrl + →`    | `Cmd + →` |

**注意事项：**

- 需要先切换到 Reading mode
- 可以在配置中自定义快捷键

---

### 📚 书架管理

集中管理你的所有书籍。

**功能列表：**

#### 导入书籍

1. 点击书架视图的 ➕ 按钮
2. 选择 `.txt` 文件
3. 等待导入完成

#### 删除书籍

- 在书架中右键点击书籍
- 选择删除选项

#### 刷新书架

- 点击 🔄 按钮刷新书籍列表
- 自动同步书籍状态

#### 切换书籍

- 直接点击书架中的其他书籍
- 会自动保存当前阅读进度

---

### 🔄 模式切换

两种模式自由切换，避免快捷键冲突。

#### Reading Mode（阅读模式）

- 启用阅读快捷键
- 适合专注阅读时使用
- 状态栏显示 📖 图标

#### Coding Mode（编码模式）

- 禁用阅读快捷键
- 不影响正常编码
- 状态栏显示 💻 图标
- **默认模式**

**切换方式：**
点击状态栏的模式按钮即可切换。

**自动切换：**
当你停止阅读时，会自动切换回 Coding mode。

---

### 💾 进度保存

自动保存阅读进度，随时继续阅读。

**保存时机：**

- 每次翻页时自动保存
- 切换书籍时保存
- 关闭 VSCode 时保存

**恢复进度：**
再次打开书籍时，会自动跳转到上次阅读的位置。

---

### ⏭️ 快速跳转

快速跳转到指定位置。

**跳转到页码：**

1. 点击状态栏的 **跳转** 按钮
2. 输入目标页码
3. 按回车确认

**跳转到百分比：**
计划中的功能，敬请期待。

---

### ⏰ 自动停止

设置无操作后自动隐藏阅读内容。

**使用场景：**

- 领导突然靠近时自动隐藏
- 长时间不操作自动停止
- 提高MO的安全性

**配置方法：**
在设置中修改 `ghostReader.autoStopDelay` 参数。

示例：

```json
{
  "ghostReader.autoStopDelay": 30 // 30秒后自动停止
}
```

详见[配置选项](./configuration.md)。

---

简洁美观的用户界面。

## 高级功能

### 多行显示

可以在状态栏同时显示多行内容。

**配置方法：**

```json
{
  "ghostReader.displayLines": 3 // 显示 3 行
}
```

**适用场景：**

- 阅读诗歌时保持格式
- 一次看到更多内容
- 提高阅读连贯性

---

### 自定义快捷键

除了默认快捷键，你还可以自定义。

**操作步骤：**

1. 打开 VSCode 设置
2. 搜索 "Keyboard Shortcuts"
3. 搜索 "GhostReader"
4. 修改对应的快捷键

**可自定义的命令：**

- `GhostReader.prev` - 上一页
- `GhostReader.next` - 下一页
- `GhostReader.jump` - 跳转
- `GhostReader.switchReadingMode` - 切换到阅读模式
- `GhostReader.switchCodingMode` - 切换到编码模式
