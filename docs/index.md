## 快速安装

### 从 VSCode 市场安装

::: code-group

```bash [命令行]
code --install-extension wllzhang.GhostReader
```

```text [VSCode]
1. 按 Ctrl+P / Cmd+P
2. 输入 ext install wllzhang.GhostReader
3. 按回车安装
```

:::

### 从 VSIX 安装

```bash
code --install-extension GhostReader-2.0.0.vsix
```

## 为什么选择 GhostReader？

- ✅ **隐蔽性强** - 在状态栏阅读，不影响工作界面
- ✅ **操作便捷** - 快捷键翻页，效率更高
- ✅ **功能完善** - 书架管理、进度保存、页码跳转一应俱全
- ✅ **体验优雅** - 简洁设计，不干扰工作流
- ✅ **开源免费** - MIT 许可证，完全开源

## 支持的平台

- Windows 10/11
- macOS 10.15+
- Linux

## 系统要求

- VSCode >= 1.95.0
- 支持 UTF-8 编码的 txt 文件
